Services I Provide.

Front-end Development.
The internet is a rapidly evolving subject. In a matter of months something can go from new, to old and outdated. This does apply to websites. Having a good developed site might look bad in a few months or even a year. Instead of following website trends for web development, I focus on themes that are timeless. By using modern frameworks that work on a vast amount of devices, to frameworks that provide a smooth UX I can deliver the smooth experience that the user expects, and that you (the company) want!
UX Developer.
As a UX Developer, my job is to make the best looking AND "feeling" website available! I add smooth scrolling plugins to make the user feel really relaxed. I also attempt to make fast loading pages so that the user can get the information as fast as possible.
Some tactics I use come from Microsoft. Pastel Color Schemes/Reverse Colors is what I use, since the colors usually match together.
Backend Developer.
Note: I'm only starting to learn backend development so these services may only improve with time. My backend is mostly just requests. Send information to 'x' with validation from 'y'. In other terms, I can do forms.
